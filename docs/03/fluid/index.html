数理科学 I / 第 3 回

局所的な法則から全体を記述する — 2 次元流体デモ

各格子点での局所的な更新を，全体に同じ形で繰り返すと，流れ全体のふるまいが現れます． このデモでは，染料の模様だけでなく，速度場・発散・圧力の表示を切り替えて， 「近くの値との差を使った更新」が全体の運動につながる様子を観察できます．

見どころ

このデモは流体力学を詳しく学ぶためのものではなく， 「偏微分で書かれた局所的な法則が，全体のふるまいを決める」ことを直感的に見るためのものです．

1. 速度場

各点に小さなベクトルがある

まず，各格子点には「その場所でどちら向きに，どれくらい流れているか」という局所情報が入っています． これが速度場です．

2. 発散

周囲との差をとる

近くの格子点との違いを比べると，その点で流れが湧き出しているか，吸い込まれているかを調べられます． これは差分で書かれる局所計算です．

3. 圧力補正

局所的に直すと，全体が整う

発散が大きい場所を，近傍の情報から少しずつ補正すると， 全体として滑らかな流れへ近づいていきます．

4. 染料表示

全体の模様として見える

局所的な更新を続けた結果が，目に見える大域的な模様として現れます． これが「局所的な法則から全体を記述する」という見方です．

局所計算の見方
近くのセルとの違いを使って，局所的に状態を更新します．

速度矢印は，内部で計算されている速度場を粗い格子で抜き出して表示したものです． 発散モードでは赤が「湧き出し」，青が「吸い込み」を表します． 圧力モードでは，発散を小さくするための補正量の分布を見ています．

このデモで意識してほしいこと

局所 → 全体

一つ一つの式は近傍だけを見る

各更新式は，自分の近くの値だけを使います．それでも，その計算を全体に一斉に適用すると， 画面全体の流れが決まります．

偏微分方程式の数値化

微分は差分として実装される

実際の計算機では，偏微分は格子上での差分として近似されます． つまり，微分の考え方が，そのまま計算アルゴリズムになります．

並列計算

同じ局所ルールを全点で同時に適用できる

各格子点で行う計算の形はほぼ同じなので，GPU のような並列計算と相性が良いです． このデモもブラウザ上で GPU を使って計算しています．

授業との接続

微分は変化の局所的な法則を表す

微分は，その点の近くでどのように変化するかを記述します． その局所法則を集めることで，現象全体を理解できることがあります．
