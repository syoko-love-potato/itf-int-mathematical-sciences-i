テイラー展開を観察する

f(x)=sin x （黒）に，x=a の近くでテイラー展開して得られる多項式 T_n(x) （青）を重ねて表示します． スライダで展開する中心 a と次数 n を変えると， 次数を上げるほど近似精度が高まる様子を観察できます． 特に a=0 のときは，マクローリン展開になっています．

範囲は -2π から 2π です．黒が sin x，青がテイラー多項式です．赤い縦線が展開の中心 x=a を表します．
次数を上げると，使う導関数の情報が増え，x=a の近くでより精密に近似できます．
現在の近似の見方
T_n(x) = \sum_{k=0}^{n} f^{(k)}(a) / k! · (x-a)^k
ここに現在の係数表示が入ります．
展開の中心
a = 0.000000
π との比
a / π = 0.000000
現在の次数
n = 1
中心での値
sin(a) = 0.000000
拡大図での最大誤差
max |sin x - T_n(x)| = 0.000000

全体のグラフ

黒線が sin x，青線が x=a のまわりでのテイラー多項式 T_n(x) です． 局所近似は展開中心の近くではよく合いますが，遠くまで行くと一致しないことがあります．

x=a 付近の拡大図

拡大図では，展開中心の近くで黒線と青線がどれほど重なるかを詳しく観察できます． 次数を上げると，一致する範囲が広がる様子が見えてきます．

拡大区間での誤差

下のグラフは，拡大図と同じ区間での誤差 |sin x - T_n(x)| です． 近似がよくなるほど，誤差の曲線は下へ押し下げられます．

テイラー展開は，関数の値だけでなく，その点での傾きや曲がり方などの導関数の情報を集めて， 関数を局所的に多項式で置き換える方法です． 離れた場所まで必ずよく合うわけではありませんが，近くで複雑な関数を扱いやすい式に置き換えられることが大きな魅力です．
