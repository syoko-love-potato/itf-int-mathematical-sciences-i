偏微分を観察する

二変数関数 z = f(x,y) = 4 + 0.5x + 0.3y + sin(x)cos(y) を例に， 片方の変数を固定して一変数関数として見ることの意味をとらえます．

2D 断面図
下の断面図を，赤または青のどちらで見るかを切り替えます．
現在の点
(a,b) = (0.80, 0.60)
関数の値
f(a,b) = 5.17
x 方向の偏微分
∂f/∂x = 1.080.5 + cos(a)cos(b)
y 方向の偏微分
∂f/∂y = -0.110.3 - sin(a)sin(b)

曲面と断面

薄い長方形は z=0 平面上に描いた定義域です． 曲面上の黒い点を通る赤・青の線が，偏微分で見る一変数関数のグラフに対応します．

断面を一変数関数として見る

y=b を固定すると，z=f(x,b) という一変数関数になります．その接線の傾きが ∂f/∂x(a,b) です．
