指数関数と対数関数の極限

底 a を動かすと，指数関数 y = ax と対数関数 y = logax の形がどう変わるかを観察できます． とくに，0<a<1 のときと a>1 のときで， x を大きくしたとき・0 に近づけたときのふるまいが逆になることに注目してください．

指数関数 y = ax

底 a が 1 より大きいと右上がり，0 と 1 の間にあると右下がりになります． 情報欄では，x→∞ と x→−∞ のときのふるまいを確認できます．

底は正で，しかも 1 ではない必要があります．
a = 2.00
現在の底
2.00
x を大きくすると
∞ に大きくなる
x を小さくすると
0 に近づく

対数関数 y = logax

対数関数は x>0 でのみ定義されます． 底 a が 1 より大きいと増加し，0 と 1 の間にあると減少します． x→0+ や x→∞ のときのふるまいにも注目してください．

底は正で，しかも 1 ではない必要があります． 1 に近いほど，グラフの傾きが大きく変わります．
a = 2.00
現在の底
2.00
x を大きくすると
∞ に大きくなる
x を 0 に近づけると
−∞ に小さくなる

スライダを動かして，0<a<1 と a>1 の両方を試してみてください． 指数関数と対数関数で，極限の向きが対応していることも確認できます．
