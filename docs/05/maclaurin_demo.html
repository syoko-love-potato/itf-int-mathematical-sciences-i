マクローリン展開を観察する

x=0 のまわりで作るマクローリン多項式 M_n(x)（青）を， 元の関数（黒）に重ねて表示します． このデモでは展開の中心は常に 0 とし，その代わりに関数の種類と次数 nを変えられるようにしました． e^x，sin x，cos x のように広い範囲でよく近似できる例に加え， 1/(1-x) のように 0 の近くではよく合う一方で遠くでは合いにくくなる例も見られます．

x=0 付近の拡大図

マクローリン展開は本来，0 の近くで関数を扱いやすい多項式で近似するための道具です．

マクローリン展開は，テイラー展開のうち展開の中心を 0 に固定したものです． 展開する点が決まっているぶん，式は (x-0)^k = x^k となってかなり見やすくなります． 一方で，よく合うのはあくまで 0 の近くです． 関数によっては，次数を上げるとかなり広い範囲で使える一方，遠くでは急に合いにくくなることもあります．
